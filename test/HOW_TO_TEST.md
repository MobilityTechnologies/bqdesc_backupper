## How to Test

cd root directory

```
cd bqdesc_backupper
```

make config

```
cp src/conf/config_sample.py src/conf/config_for_test.py
vim src/conf/config_for_test.py
```

make tables

```
test/make_test_dataset_and_tables.sh
```

run test

```
python -m unittest discover test "test_*.py"
```

run linter

```
tox -e lint
```
